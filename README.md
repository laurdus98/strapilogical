# strapilogical

Understanding Strapi's CMS logical

## how to install

### open your terminal

#### git clone link-repo

#### cd strapilogical

#### cd api

#### yarn

#### yarn develop

#### listen port http://localhost:1337/admin/local

#### open other terminal project strapilogical cloned

#### cd strapi-apollo

#### yarn && yarn dev

#### listen port http://localhost:3000
